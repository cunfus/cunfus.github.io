169 - 多数元素

数组

给定一个大小为 n 的数组 nums ，返回其中的多数元素。多数元素是指在数组中出现次数 大于 ⌊ n/2 ⌋ 的元素。

解题思路：

  1. sort and fetch the middle element.
  2. traverse nums, use map store counts; traverse map, found max counts.
