栈

stack 是一个 先入后出（FILO, First In Last Out） 的 有序列表。

场景

栈 常见的应用场景，

  • 函数调用，C 里的 stack frame 栈帧，Python 里的 global/local frame 全局/局部帧。
  • 表达式转换，中缀表达式转后缀表达式， (3 + 4) * 5 -> 3 4 + 5 *。
  • 二叉树的遍历，DFS（回溯）
  • 常见的回溯算法，尝试（append） 回退（pop） 剪枝（if not）

操作

用 Python 的 list 来模拟栈

stack = []       # 空栈
stack.append(1)  # 入栈
stack.pop()      # 出栈
stack[-1]        # 访问栈顶元素
len(stack)       # 获取栈长度
len(stack) == 0  # is_empty?
