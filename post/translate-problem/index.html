再说翻译问题

台版翻译比大陆翻译好很多，为什么会这样？

前段时间，我读过一本 Deep Work。

当时分别下载了英文、大陆翻译和台湾翻译三种版本。我开始读的是大陆翻译的版本，感觉像是回到了盛行的鸡汤文学。怀疑译者夹杂了私货，于是我对照英文原版观察，可意思也能说的通。再对照台版翻译，简中的译文就粗糙了许多。

那么台版翻译好在哪呢？仅看目录，

  • 简化
    • 从 xxx 角度论证 -> 从 xxx 来看
  • 清楚
    • 经商 -> 经营企业
    • 工作要深入 -> 培养深度工作力
    • 习惯化 -> 建立仪式
    • 要有大手笔 -> 大动作投入
    • 变得不容易联系到 -> 让自己难以联络
  • 合理
    • 图安逸 -> 要懒惰
    • 不要不断分心，而要不断专注 -> 安排分心的时间
    • 像罗斯福一样工作 -> 像罗斯福那样工作
    • 记住一副牌 -> 练习记忆一副牌

语言本来是用来交流的，同样是中文，为什么简中版本会如此…乏味？

  • 行文缺乏主语，模糊就是让对面去猜测，是语言暴力
  • 语言没有原作的韵味和情感
  • 审查制度，它影响到了某些内容的翻译和呈现。但对于有心的译者来说，这不是主要因素

简中似乎已经丧失了很多语言特有的细腻和表达的准确性。

之前阅读一本技术书，总结了中文版不如英文原版（这里）。现在看，应该是 原书 > 台版 >> 简中。
