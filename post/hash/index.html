哈希

Hash 哈希，建立键 key 和值 value 之间的映射，实现高效的元素查询。

场景

哈希 常见的应用场景

  • 完整性检查，md5 sha
  • 缓存，用来同步信息
  • 密码存储，不存明文密码，比对哈希值是否一致

操作

hash = {}                  # 空哈希表
hash['key'] = 'value'      # 增加 key-value
hash.pop('key')            # 删除 key-value
hash.get('key')            # 查询 value
hash.clear()               # 清空哈希表

# 遍历
for key, value in hash.items()
for key in hash.keys()
for value in hash.values()
