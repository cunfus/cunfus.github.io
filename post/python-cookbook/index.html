Python Cookbook

  1. 选择特定的变量，作为丢弃的值的名称。
data = [ 'ACME', 50, 91.1, (2012, 12, 21) ]
_, shares, price, _ = data
  1. *表达式用于 unpack 过程
grades = [96, 87, 77, 65, 56]
first, *middle, last = grades
  1. defaultdict 一键多值字典
from collections import defaultdict
d = defaultdict(list)
d['a'].append(1)

可以避免自己写 list 的初始化

  1. OrderedDict字典序与序列化
from collections import OrderedDict
d = OrderedDict()
d['foo'] = 1
d['spam'] = 3
d['bar'] = 2

for key in d:
    print(key, d[key]) 

import json
print(json.dumps(d))

构建一个控制顺序的映射结构，然后序列化，是比较常用的。
