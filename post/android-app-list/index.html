Android 常用软件清单

谷歌系列

微软系列

  • Github 托管平台，托管啥的都有
  • Authenticator 配合 Github 验证
  • ToDo 待办清单，配合桌面的 Ao 同步使用

其他

  • F-Droid 自由开源的 android 软件库

  • Poe 综合各种模型的应用平台
  • Spotify 听音乐
  • Telegram 隐私程度更高的即时通讯

更新

尽量保持跟手机的距离。生活中是需要用它购物、沟通、出行、看天气，即使是这些基础的功能，尽量减少使用频率。用手机消遣无聊，往往是灾难的开始，留给自己的时间尽量越多越好。
