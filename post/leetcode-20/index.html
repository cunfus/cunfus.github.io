20 - 有效的括号

栈

给定一个只包括 ‘(’，’)’，’{’，’}’，’[’，’]’ 的字符串 s ，判断字符串是否有效。 有效字符串需满足：

  1. 左括号必须用相同类型的右括号闭合。
  2. 左括号必须以正确的顺序闭合。
  3. 每个右括号都有一个对应的相同类型的左括号。

解题思路：

def isValid(self, s: str) -> bool:
    stack = []
    
    for c in s:
        if len(stack) == 0:
            stack.append(c)
        elif c == ')' and stack[-1] == '(':
            stack.pop()
        elif c == ']' and stack[-1] == '[':
            stack.pop()
        elif c == '}' and stack[-1] == '{':
            stack.pop()
        else:
            stack.append(c)

    return len(stack) == 0
