Disc-使用 k3b

New 新建

数据盘

  • New Data Project 从空白盘开始
  • Continue Multisession Project 用刻录过的剩余光盘空间继续

DVD-Video

  • DVD-Video 应包含标准的文件结构：
    • VIDEO_TS 文件夹：包含 .VOB、.IFO 和 .BUP 文件
    • AUDIO_TS 文件夹：通常为空，但应存在
  • 一般采用较低的刻录速度（4x），保证刻录质量

CD-DA

  • 支持多种音频格式，WAV、MP3、OGG、FLAC 等会转换成音频 CD 需要的格式
  • 可以设置音轨之间的间隔，调整播放体验

Mixed CD

New Mixed Mode CD Project 混合模式 CD，可以在同一张 CD 上同时存储音频轨道和数据文件。音频轨道通常在光盘的前面，而数据文件则在后面。它在大多数 CD 播放器中播放音频部分，而计算机可以读取数据部分，适合需要同时分享音乐和相关数据（如歌词、封面、说明文档等）的场景。

Burn 刻录

  • Writing Mode
    • AUTO 自动模式，根据光盘的类型和内容自动选择最合适的写入模式
    • DAO（Disc At Once）模式，整个光盘在一次写入过程中被刻录完成，不能再进行任何写入或修改
  • Setting
    • Simulate，模拟刻录过程，不实际写入光盘，适合检查设置
    • Create Image，创建数据映像文件并准备刻录，适合保留副本
    • Only Create Image，仅创建映像文件，不进行刻录，适合只需备份数据的情况
    • Verify Written Data，刻录重要文件时，可以确保数据的可靠性和完整性
  • FileSystem
    • Volume Name，最大 16 个字符
    • 其他的默认就行
