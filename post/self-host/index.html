Self-Hosted

出于数据可控的目的，折腾过一些自托管应用，顺带了解了 VPS，CyberPanel，域名，Docker。

  • memos 卡片笔记，flomo 的开源版本
  • freshrss 订阅 rss
  • archivebox 文章归档
  • jupyter 在线 Python 环境
  • coder vscode 的在线版本
  • gitweb 可视化代码仓库
  • gitea 软件开发版本控制平台
  • calibre-web 在线阅读器
  • wordpress 博客平台
  • task todo 清单

用了半年以上，并没有感受到自由，精力放在了维护 VPS 和备份数据上。calibre 有了，却不看书，不知道看什么书。memos 有了，记下的东西很零碎，也没有养成习惯总结，产出了一堆腐烂数据。wordpress 平台有了，写不出内容。git 仓库私有了，基本上也是个备份，没有动力 push。

目前已回到专有方案。

最初想要数据可控，折腾了一圈，手段变成了目的。引以为戒！
