Disc-磁盘刻录基础

蓝光光盘，Blu-ray Disc，缩写为 BD，比常规的 DVD 光盘容量更高。

类别

  • BD
    • BD-R 单层，容量 25GB
    • BD-R DL Dual Layer，容量 50GB
    • BD-R XL eXtended Layer，三层的 100GB，四层的 128GB，五层的 200GB
  • DVD
    • 单层容量 4.7GB
    • 双层容量 8.5GB
  • CD
    • 容量 700 MB
    • CD-Audio 专用于存储音频
    • CD-ROM 存储任何类型数据

写入速度

对于 DVD 以上，1x 的写入速度是 36 Mbps， 16x 是 576 Mbps。单层和双层的一般是 116x，多层的一般在 16x。速度越快，完成的时间越短。

对于 CD，1x 的写入速度是 150 Kbps，52x 是 7.8MBps。

尺寸

  • CD DVD BD 光盘通用尺寸（直径）是 120mm，厚度 1.2mm
  • 小型光盘（冷门）通用尺寸是 80mm，CD 容量 200MB，DVD 容量单层 1.4GB，双层2.8GB

兼容

播放机器向下兼容，UHD > BD > DVD > CD，UHD 的机器能播放 UHD、BD、DVD 和 CD 光盘，BD 的机器能播放 BD、DVD 和 CD 光盘，DVD 的机器能播放 DVD 和 CD 光盘，CD 的机器只能播放 CD 的光盘了。

播放 UHD 蓝光光盘时，需要具备4K UHD电视以及相应的 HDMI 线缆，以支持更高的分辨率和视频质量。

刻录软件

  • windows : ImgBurn
  • linux: k3b + dvd+rw-tools
