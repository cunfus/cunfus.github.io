88 - 合并两个有序数组

数组 排序

给你两个递增整型数组 nums1 和 nums2 元素数量分别是 m 和 n。 将 nums2 合并到 nums1，保持 nums1 仍为递增序列。

解题思路：

从后往前遍历 nums1 和 nums2， 将较大的元素放到 nums1 的最后。

class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int):
        """
        Do not return anything, modify nums1 in-place instead.
        """
        i, j, k = m - 1, n - 1, m + n - 1
        
        while i >= 0 and j >= 0:
            if nums1[i] > nums2[j]:
                nums1[k] = nums1[i]
                i -= 1
            else:
                nums1[k] = nums2[j]
                j -= 1
            k -= 1
        
        # 如果 nums2 还有元素未合并，直接添加到 nums1 的前面
        while j >= 0:
            nums1[k] = nums2[j]
            j -= 1
            k -= 1
