# Timewarrior

!!! abstract
    律道杀招——井井有条，通过命名行跟踪记录时间。

一个事后总结分析工具。

## 基本功能

- `timew start` 开始 time tracking
- `timew stop`  停止 time tracking
- `timew continue` 继续 time tracking
- `timew summary` 查看摘要

## 更多功能
- `timew tags` 查看标签
- `timew track` 事后记录一段时间
- `timew day/week/month` 查看图表报告





## 集成

Timewarrior 可以通过 Taskwarrioron-modify 挂钩[脚本](https://timewarrior.net/docs/taskwarrior/)与 Taskwarrior 集成。

安装后，每当任务处于活动状态时，Timewarrior 都会用于自动跟踪时间。
