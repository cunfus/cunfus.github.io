队列

queue 队列是一个 先入先出（FIFO, First In First Out） 的 有序列表。

场景

队列 常见的应用场景

  • 消息队列
  • 二叉树的遍历， BFS
  • 业务逻辑解耦，划分模块，共享同一个队列的访问
  • 循环缓冲区，字节流的网卡收包，数据流的压缩算法，结合链表的时间轮

操作

用 Python 的 list 来模拟队列

queue = []          # 空队
queue.insert(0, 1)  # 入队
queue.pop()         # 出队
queue[0]            # 访问队顶元素
len(queue)          # 获取队长度
len(queue) == 0     # is_empty?
