Manjaro 常用软件清单

上网

  • Clash 科学上网，一切的开始
  • Nekoray 备用

办公

  • Ao 微软系列，包括 ToDo Outlook
  • Insync 网盘系列，包括 OneDrive
  • Chrome 如果不是插件太好用，就转向 Firefox 了

编码

  • Helix 专门写 C/C++ 的，开箱体验比 VSCode 舒服
  • VSCode 专门写 Python 的，毕竟插件多了会卡，会冲突，很烦
  • Meld 代码比对

本地

  • Konsole 默认终端
  • Fcitx5 小企鹅输入法，不会偷偷在后台运行别的
  • KDE Connect 连接手机，共享 copy-paste
  • LocalSend 局域网传输，超大文件效果不太好
  • Foliate 查看文档，对 epub mobi 支持较好
  • Okular 标注 pdf
  • Flameshot 截图
  • LibreOffice 文档处理
  • Typora 处理 markdown，准备弃用，转向 asciidoc
  • Kate 纯文本类文档编辑

其他

  • MineCraft 创世之神
  • Telegram 注重隐私的即时通讯

chrome 插件

  • Bitwarden Password Manager 不用记密码
  • Language Reactor 配合 youtube 使用，看字幕熟悉英文
  • OneTab 浏览器内存救星
  • Tampermonkey 发现 RSS 链接
  • Zotero Connector 网页文档保存
